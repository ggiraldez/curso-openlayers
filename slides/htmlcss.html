HTML y CSS

HTML

  • La idea es separar el contenido (el qué) de la presentación (cómo se muestra)
  • HTML provee etiquetas que le dan semántica al contenido
    • head, title, body
    • h1 a h6
    • p, img
    • table, thead, tbody, tr, th, td
    • ul, ol, li
    • div, span
  • El navegador tiene un estilo predeterminado para cada etiquetas, pero luego lo podemos modificar con una hoja de estilos (CSS)

HTML - Hypertext Markup Language

Estructura básica de un documento

<!doctype html>
<html>
  <head>
    <meta charset="utf-8">
    <title>Título de la página</title>
  </head>
  <body>
    <h1>Título</h1>
    <p>Esto es un párrafo
  </body>
</html>

HTML

Atributos comunes a todos los elementos

  • id="boton1" debe ser único en el documento
  • class="listado primario" se usa para vincular con CSS, pero se puede usar semánticamente
  • style="color: red;" se usa para embeber CSS en un elemento particular
  • data-* datos de usuario (eg. data-parametro="algún dato arbitrario")

HTML

  • HTML no es estricto como XML
  • Existe XHTML, pero está siendo depreciado
  • Algunas etiquetas de cierre son opcionales
  • No es necesario encomillar el valor de los atributos
  • Entidades HTML (& + codigo + ;)
    • para representar caracteres especiales
    • &lt; &gt; &amp; &quot;
    • &copy; &aacute; &ntilde; &euro;
    • &#numero;
  • Hay caracteres especiales que pueden codificarse directamente
  • Para eso hay que indicar el character encoding del documento
    • tag <meta charset="utf-8">
    • encabezado HTTP: Content-type: text/html; charset=utf-8
    • tag <meta http-equiv="content-type" content="text/html; charset=utf-8">

HTML

Disposición (layout) de elementos

  • Hay 4 grandes tipos de layout
    • inline
    • block
    • list-item
    • table
  • Y también está float
  • Cada tag de HTML tiene asociado un tipo predeterminado de layout
  • Pero puede alterarse mediante CSS

CSS - Cascading Stylesheets

Ejemplo

body {
  margin: 0;
  padding: 0;
}
h1 {
  color: blue;
  background-color: #ffff00;
  font-size: 18px;
}
img {
  border: 1px solid green;
}
p {
  margin-top: 5px;
}

CSS - Cascading Stylesheets

Selectores

  • Selector universal *
  • Por tag DIV
  • Por clase .listado
  • Por ID #boton1
  • Pseudo-clases :hover
  • Por atributos input[type=text]
  • Combinación de selectores
    • Descendiente (E F)
    • Descendiente directo (child) (E > F)
    • Hermano (E + F)

Posicionamiento

  • top, bottom, left, right
  • width, height
  • position
    • static es el posicionamiento predeterminado
    • relative el elemento no se quita del flow, pero se puede desplazar
    • absolute el elemento se quita completamente del flow y se posiciona según el ancestro más cercano posicionado
    • fixed idem anterior, pero la posición es relativa a la ventana
